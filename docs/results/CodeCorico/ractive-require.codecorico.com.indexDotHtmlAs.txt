aria-hidden="true" class="anchor" href="#what-is-it" id="what-is-it">\n       <span class="icon-link">\n       </span>\n      </a>\n      What is it?\n     </h2>\n     <blockquote>\n      <p>\n       <em>\n        <a href="http://http://www.ractivejs.org/" target="_blank">\n         Ractive.js\n        </a>\n        is a template-driven UI library, but unlike other tools that generate inert HTML, it transforms your templates into blueprints for apps that are interactive by default.\n       </em>\n      </p>\n     </blockquote>\n     <p>\n      Ractive-Require is an extension to Ractive.js that provides the support to require features on demand. A feature is composed by a HTML template file, and its JavaScript and CSS assets. They are not loaded on the page as long as the user does\'t need to use it.\n     </p>\n     <p>\n      The final goal is to create large webapps that loads and uses features only when necessary.\n     </p>\n     <!-- THE PROJECT -->\n     <h2>\n      <a aria-hidden="true" class="anchor" href="#project" id="project">\n       <span class="icon-link">\n       </span>\n      </a>\n      The project\n     </h2>\n     <div class="row">\n      <div class="two columns">\n       <img alt="CodeCorico" src="/assets/CodeCorico.png"/>\n      </div>\n      <div class="ten columns">\n       <p>\n        The project is maintened by an enthusiast community.
